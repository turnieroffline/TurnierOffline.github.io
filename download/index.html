ANDROID / TERMUX
Direkt auf dem Handy · Hotspot für alle Geräte
📱 ALTERNATIVE
1
Termux installieren
Aus dem F-Droid Store herunterladen (nicht Google Play — veraltete Version!).
2
ZIP herunterladen & Setup-Script ausführen
ZIP in den Termux-Ordner legen, entpacken und Script starten:
bash termux_setup.sh
3
WLAN-Hotspot aktivieren
Handy-Hotspot einschalten. Andere Geräte (TV, Laptops) verbinden sich damit. TurnierOffline läuft unter http://192.168.43.1:8080
4
Setup Wizard öffnen
Beim ersten Aufruf von /admin erscheint der Setup Wizard — Passwort & Turniername einrichten, fertig.
Android 7.0+ ab ca. 2017 300 MB frei Termux (F-Droid) ⚠ Firewall kann Ports blockieren
🪟
WINDOWS
XAMPP · einfachstes Setup
👍 GUT GEEIGNET
1
XAMPP installieren
apachefriends.org — PHP 8.2 Version wählen. Nur Apache & PHP auswählen reicht.
2
ZIP entpacken & Script starten
ZIP nach C:\xampp\htdocs\turnier\ entpacken, dann:
windows_setup.bat
3
Windows-Firewall freigeben
Windows fragt beim ersten Start nach — Apache den Netzwerkzugriff erlauben. Dann sind alle Geräte im selben WLAN erreichbar.
4
Setup Wizard öffnen
Browser: http://localhost/admin — Setup Wizard erscheint beim ersten Aufruf automatisch.
Windows 10 / 11 ab ca. 2016 1 GB frei XAMPP (PHP 8.2)
🐧
LINUX
Ubuntu · openSUSE · systemd · stabiler Dauerbetrieb
⭐ EMPFOHLEN
1
ZIP herunterladen & entpacken
ZIP auf den Linux-Rechner kopieren (z.B. per USB oder wget), dann entpacken.
unzip TurnierOffline_V16.zip
2
Setup-Script ausführen
Installiert PHP, SQLite und richtet Apache oder den eingebauten PHP-Server ein:
sudo bash linux_setup.sh
3
Autostart einrichten (optional)
Das Script fragt ob ein systemd-Service eingerichtet werden soll — dann startet TurnierOffline automatisch beim Booten.
4
Setup Wizard öffnen
Browser: http://localhost/admin — bei Bedarf IP-Adresse des Rechners für andere Geräte im Netz nutzen.
Ubuntu 20.04+ ab ca. 2013 512 MB RAM PHP 8.0+ SQLite 3
🐳
DOCKER
Plattformunabhängig · isoliert · 50 MB
🔧 FÜR PROFIS
1
Docker installieren
docs.docker.com — Docker Desktop (Win/Mac) oder Docker Engine (Linux).
2
ZIP entpacken & starten
ZIP entpacken, in den Ordner wechseln und starten:
docker compose up -d
3
Setup Wizard öffnen
Browser: http://localhost:8080/admin
Docker Engine 20+ Win / Linux / Mac 1 GB RAM ⚠ Docker-Kenntnisse nötig
