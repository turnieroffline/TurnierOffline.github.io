Features

ALLES DABEI

Alles was du für einen Turniertag brauchst — auf deinem Gerät, ohne Internet, ohne Abo.

📺
TV-ANZEIGE
Automatischer Seitenwechsel. Spielstand, KO-Bracket, Tabelle, Werbung — alles konfigurierbar.
Auto-Zyklus
⚽
LIVE-EINGABE
Tore, Karten, Auswechslungen — ein Tipp und alle sehen es sofort. Torschütze und Spielminute.
Echtzeit
📱
HANDY-APP
Zuschauer verbinden sich per QR-Code. PWA — kein Download nötig. Jeder Browser.
PWA
🏆
KO-RUNDEN
Losziehung per Knopfdruck. Automatischer Spielbaum. Finale bis Viertelfinale.
Automatisch
🤖
KI-ASSISTENT
Zuschauer fragen: „Wer spielt gerade?" — regelbasiert oder Ollama, auch ohne Internet.
Offline KI
🔊
SPRACHANSAGEN
Automatische Ansagen bei Toren, Karten, Halbzeit. Stimme und Lautstärke einstellbar.
Text-to-Speech
🏟
MULTI-FELD
Mehrere Spiele gleichzeitig. TV zeigt Split-Screen. Bis zu 4 Felder gleichzeitig.
Bis 4 Felder
🌐
GITHUB SYNC
Nach jedem Tor automatisch auf GitHub Pages gepusht — für Fans die zuhause mitfiebern.
Auto-Push
🎯
TIPP-SPIEL
Zuschauer tippen Ergebnisse. 3 Punkte exakt, 1 Punkt Sieger. Live-Rangliste.
Neu V14
📵
100% OFFLINE
Läuft komplett lokal. Kein Internet, kein Server, kein Abo. Alles auf deinem Gerät.
Kein Cloud
🔌
WORDPRESS
Plugin zum Einbetten des Live-Stands auf der Vereinswebseite. Automatisches Update.
Plugin enthalten
🏅
SIEGEREHRUNG
Automatisches Podest mit allen Ergebnissen. Export als HTML für das Vereinsprotokoll.
HTML Export
